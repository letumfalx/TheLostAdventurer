# TheLostAdventurer
A simple platformer game using PIXI JS.
It has 5 levels with increasing difficulty.
Enjoy!

Working game link:
http://mygame.likesyou.org/games/thelostadventurer


Controls:

left    - going left

right   - going right

up      - entering door

space   - jump
